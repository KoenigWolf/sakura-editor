現在の変更または最新のコミットをGitHub用のタイトルと説明に要約します。

## 手順

1. `git status` で変更の有無を確認
2. 変更がある場合は `git diff` を確認、なければ `git log -1` と `git show` で最新コミットを確認
3. 変更内容を分析

## 出力形式

以下の形式で出力:

```
## Title
<50文字以内の簡潔なタイトル（英語）>

## Description
<変更内容の説明（日本語）>

### 変更点
- 変更1
- 変更2
- ...

### 影響
- 影響範囲や注意点があれば記載
```

## ルール

- タイトルは英語で、コミットメッセージのような簡潔な形式
- 説明は日本語で、PRやIssueの説明として使える詳細さ
- 技術的な変更は具体的に記載
- 破壊的変更がある場合は明記
